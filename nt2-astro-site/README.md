# NT2 website starter

Starter in Astro voor een NT2 website van Kristina Pichugina.

## Installeren

```bash
npm install
npm run dev
```

Open daarna de lokale preview URL die Astro toont in je terminal.

## Belangrijk aanpassen

- Vervang in `src/pages/offerte-aanvragen.astro` en `src/pages/contact.astro` de `action` van het formulier door je Formspree of andere formulier-endpoint.
- Pas in `src/layouts/BaseLayout.astro` de canonical site URL aan in `astro.config.mjs`.
- Voeg later extra locatiepagina's toe in `src/pages/`.
